Share's Sweet Treats - Final Website Starter

Open index.html in your browser to preview the website.

What is included:
- Branded homepage
- Interactive menu with search and categories
- Shopping cart with quantities
- Pickup, 20-mile delivery, and mail shipping options
- Delivery fee estimate
- Payment method choices
- Order request form
- Thank-you email/text quick-send buttons
- Owner dashboard preview
- Test order storage in browser localStorage
- CSV order export for testing
- Launch checklist

Important launch note:
This static starter is ready for branding and domain setup, but real online payments and automatic messages require services such as Stripe/Square/PayPal, EmailJS/Formspree, Twilio, and a backend/database.
